# toy-browser ⚡

**从零到一完成一个玩具浏览器来了解浏览器渲染原理**


![infoJs](https://static01.imgkr.com/temp/d77a025e2eb14ef9857cdccf5015b9e2.png)

## Install 💡

```shell
git clone https://github.com/sudongyuer/toy-browser.git

npm install
```

## Useage 🌲

**需要在本地安装node环境哦，没有安装的小伙伴请在本地安装node环境**

###在src目录分别启动服务端和客户端

**step1: 启动服务端**

```shell

node server.js

```

**step2:启动客户端**


```shell

node client.js

```
